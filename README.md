## About me

I work with Linux and Kubernetes environments, focusing on practical platform operations: deploying services, managing secrets, troubleshooting infrastructure issues and documenting repeatable procedures.

My repositories are mostly hands-on labs and technical notes. The goal is not only to make something work, but to explain the reasoning behind each step.

Current focus areas:

- Kubernetes administration
- Helm chart design and deployment
- GitOps with Argo CD
- HashiCorp Vault and External Secrets
- Keycloak identity labs
- PostgreSQL on Kubernetes
- Prometheus and Grafana monitoring
- Linux troubleshooting and SRE-style exercises
